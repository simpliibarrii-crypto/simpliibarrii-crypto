## Use Cases

- **Healthcare:** Clinical note generation, drug interaction checking, patient monitoring (HIPAA/PHIPA)
- **Finance:** Local document analysis, compliance review, data extraction
- **Legal:** Confidential document processing, contract analysis
- **Government:** Air-gapped AI deployment, sovereign data processing
- **Enterprise:** Internal knowledge base, private code assistant, data analysis

---

## Deployment Options

1. **Local (single machine)** — Laptop/desktop with local models
2. **On-prem (cluster)** — Private datacenter with GPU nodes
3. **Hybrid** — Local agent with cloud fallback (policy-controlled)
4. **Air-gapped** — Fully disconnected, no external dependencies
