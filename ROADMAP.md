# Raven AI — Project Roadmap

## Current Status — v0.1.0 (Pre-release)

Laying the foundation for sovereign, local-first AI orchestration.

### Q3 2026 — Foundation
- [ ] Core orchestration engine
- [ ] Plugin architecture for AI providers
- [ ] Local-first data storage with encryption
- [ ] Cross-platform CLI tool
- [ ] CI/CD pipeline and automated testing

### Q4 2026 — Intelligence Layer
- [ ] Multi-model support (GPT, Claude, local models)
- [ ] Context management and memory
- [ ] Tool-calling and function execution
- [ ] API gateway with rate limiting and auth
- [ ] Web UI dashboard

### Q1 2027 — Enterprise
- [ ] Audit logging and compliance reporting
- [ ] Role-based access control (RBAC)
- [ ] Federated deployment across nodes
- [ ] SOC 2 / ISO 27001 readiness
- [ ] HIPAA compliance module

### Q2 2027 — Ecosystem
- [ ] Plugin marketplace
- [ ] Community templates and workflows
- [ ] Mobile companion app
- [ ] Enterprise SSO and directory integration
